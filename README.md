# `untar`

A simple utility to easily extract tar files.

## Usage
```sh
untar <filename>
```

## Supported Extensions
- `.tar`
- `.tar.gz`
- `.tar.tgz`
- `.tar.taz`
- `.tar.bz2`
- `.tar.tz2`
- `.tar.tbz2`
- `.tar.tbz`
- `.tar.xz`
- `.tar.zst`
- `.tar.tzst`
- `.tar.lzma`
- `.tar.tlz`
